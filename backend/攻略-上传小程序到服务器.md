# 小程序上传攻略 —— 宝塔面板版

## 你只需要做一件事

把本地的 `mini-program.html` 传到服务器的网站根目录，它就能和 PC 端、手机 H 端一样正常访问了。

---

## 第一步：找到本地的文件

**文件位置（在你自己电脑上）：**

```
/Users/liyong/Documents/车贷业务/backend/public/mini-program.html
```

可以直接从访达里拖出来：

> 打开访达 → 左侧 `文稿` → `车贷业务` → `backend` → `public` → 找到 `mini-program.html`

---

## 第二步：登录宝塔面板

打开浏览器，访问你的宝塔面板地址（一般长这样）：

```
http://你的服务器IP:8888
```

输入用户名和密码登录。

---

## 第三步：进入网站目录

宝塔里按这个路径点进去：

```
宝塔面板 → 左侧「文件」
    → 顶部路径框输入: /www/wwwroot/xiaomengjisu.com/
    → 回车
```

你会看到类似这样的文件列表：

| 文件名 | 说明 |
|---|---|
| `server.js` | 后端服务 |
| `index.html` | PC 端主站 |
| `landing.html` | 手机 H 端落地页 |
| `admin/` | 管理后台文件夹 |
| `public/` | 静态资源 |
| `data/` | 数据文件 |

`mini-program.html` 就要放到这个目录里，和 `index.html` 平级。

---

## 第四步：上传文件

在这个页面里，点顶部的 **「上传」** 按钮：

```
[📁 上传]  [📂 新建目录]  [🔄 刷新]
```

然后：

1. 点 **「选择文件」**
2. 在弹窗里找到刚才说的本地文件：
   `/Users/liyong/Documents/车贷业务/backend/public/mini-program.html`
3. 选中它，点 **「打开」**
4. 点 **「开始上传」**

上传进度条走完就成功了。

---

## 第五步：验证

打开浏览器，访问：

```
http://124.221.136.8/mini-program.html
```

如果看到长安鑫生风格的手机界面，底部有「首页」「推广」「我的」三个 Tab，说明上传成功。

---

---

## 万一遇到问题

| 现象 | 原因 | 解决 |
|---|---|---|
| 打开显示 404 | 文件没传到正确目录 | 确认上传到了 `/www/wwwroot/xiaomengjisu.com/` 下面 |
| 上传后立即报错 | 文件权限不对 | 在宝塔终端执行：`chown -R www:www /www/wwwroot/xiaomengjisu.com/mini-program.html` |
| 页面能打开但提交没反应 | API 地址不对 | 文件是连 `/api/submit/app` 的，同源自动通，不需要改 |
| 页面样式乱了 | 文件没传完整 | 重新上传，确认大小是 42KB |

---

## 文件信息

```
文件名:     mini-program.html
大小:       42 KB (42,434 字节)
类型:       纯静态 HTML（全部样式+JS 内嵌）
API 连接:   /api/submit/app（和 PC 端/落地页同后台）
```

上传完成后小程序和 PC 端、手机 H 端共享同一个数据库，所有提交的线索都会出现在后台「线索管理」里。
